## Codebase Patterns

- Tailwind spacing is defined under `theme.spacing` (replaces defaults), not `theme.extend.spacing`
- Color tokens use `stroke`/`fg` naming to avoid Tailwind utility class collisions (`border-stroke`, `text-fg-muted`)
- All Tailwind color classes in data files must be full strings (no template literals) — JIT purges dynamic expressions
- `@astrojs/check` and `typescript` must be installed alongside the main deps for `npm run typecheck` to work
- Minimal structure for typecheck: `src/env.d.ts`, `tsconfig.json` extending `astro/tsconfigs/strict`, one page
